Quais os tipos de folhas de Estilo do CSS. Explique cada um.








































inline - o css é colocado no elemento específico no html ex: < p style= "color: red;">
incorporado- o css é colocado no cabeçario do html ex: < head> < style> p { color: #000; } < /head>
externo - o css é colocado em um arquivo externo ex < link rel="stylesheet" href ="arquivo.css"></a>
